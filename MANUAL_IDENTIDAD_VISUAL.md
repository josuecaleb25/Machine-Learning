## Introducción

**EcoVision Smart City** es una plataforma inteligente de clasificación de residuos que integra tecnología de reconocimiento facial, inteligencia artificial y geolocalización para revolucionar la gestión de residuos urbanos.

### Valores de Marca
- **Innovación:** Tecnología de vanguardia al servicio del medio ambiente
- **Sostenibilidad:** Compromiso genuino con el reciclaje y la ecología
- **Inteligencia:** Soluciones basadas en IA y datos
- **Accesibilidad:** Herramientas fáciles de usar para todos
- **Transparencia:** Información clara y métricas visibles

### Misión
Facilitar la gestión inteligente de residuos urbanos mediante tecnología de visión por computadora, permitiendo a los usuarios clasificar correctamente sus desechos y encontrar puntos de reciclaje cercanos.

### Visión
Convertirnos en la plataforma líder de gestión inteligente de residuos en ciudades latinoamericanas, contribuyendo a la creación de ecosistemas urbanos más sostenibles.

## Logotipo

### Logotipo Principal

**Construcción:**
- **Símbolo:** Hoja digital estilizada con circuitos integrados
- **Tipografía:** Sans-serif moderna (Inter/Poppins)
- **Color primario:** Verde eco (#10b981) + Azul tecnológico (#3b82f6)

### Variantes del Logotipo

#### 1. Versión Horizontal
```
[ICONO] EcoVision Smart City
```
- **Uso:** Headers, landing pages, documentación
- **Proporción:** Ícono = 1:1, Espaciado = 0.5x altura del ícono

#### 2. Versión Vertical
```
    [ICONO]
   EcoVision
  Smart City
```
- **Uso:** Avatares, favicons, apps móviles
- **Proporción:** Centrado, texto no mayor a 1.2x ancho del ícono

#### 3. Versión Isotipo
```
[ICONO]
```
- **Uso:** Favicons 16x16, 32x32, badges
- **Tamaño mínimo:** 24px × 24px

### Área de Protección

El logotipo debe mantener un espacio libre equivalente a la altura de la letra "E" de "EcoVision" en todos sus lados.

```
     ←─ E ─→
   ┌─────────┐
E  │ [LOGO]  │ E
   └─────────┘
     ←─ E ─→
```

### Tamaños Mínimos

| Aplicación | Tamaño Mínimo |
|-----------|---------------|
| Web (header) | 180px ancho |
| Favicon | 32px × 32px |
| Redes sociales (perfil) | 400px × 400px |
| Impresión | 3cm ancho |
| Email signature | 120px ancho |

### Versiones por Fondo

#### Fondo Claro
- Logotipo en color completo
- Texto en gris oscuro (#1f2937)

#### Fondo Oscuro
- Logotipo en blanco puro (#ffffff)
- Opcional: versión con ícono en gradiente eco

#### Fondo de Color
- Evaluar contraste mínimo WCAG AA (4.5:1)
- Preferir versión monocromática blanca

### Usos Incorrectos

❌ **Prohibido:**
- Rotar el logotipo
- Cambiar proporciones (estirar/comprimir)
- Aplicar efectos de sombra o relieve
- Usar colores no autorizados
- Colocar sobre fondos que comprometan legibilidad
- Rediseñar o modificar elementos del ícono

### Reglas Tipográficas

1. **Alineación:**
   - Títulos: Centrados o izquierda según contexto
   - Cuerpo: Siempre izquierda
   - Nunca justificar texto en web

2. **Longitud de línea:**
   - Óptimo: 50-75 caracteres
   - Máximo: 90 caracteres

3. **Énfasis:**
   - Usar peso bold (600-700) para resaltar
   - Evitar subrayado excepto en enlaces
   - Usar color para diferenciar categorías

4. **Capitalización:**
   - Títulos: Sentence case (solo primera palabra)
   - Botones: Sentence case
   - Navegación: Sentence case
   - Evitar ALL CAPS excepto en labels pequeños

### Fuentes Fallback

```css
/* Stack completo */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', 
             'Helvetica Neue', Arial, sans-serif, 'Apple Color Emoji', 
             'Segoe UI Emoji', 'Segoe UI Symbol';
```

**Fuente principal:** Lucide Icons / Heroicons

**Licencia:** MIT

### Especificaciones

#### Tamaños Estándar

| Uso | Tamaño | Stroke Width |
|-----|--------|--------------|
| Iconos pequeños | 16px | 2px |
| Iconos UI estándar | 24px | 2px |
| Iconos grandes | 32px | 1.5px |
| Iconos hero | 48px+ | 1.5px |

#### Colores de Iconos

- **Activos:** Verde eco (#10b981) o Azul (#3b82f6)
- **Inactivos:** Gray 400 (#9ca3af)
- **Sobre fondo oscuro:** Blanco (#ffffff)
- **Categorías:** Usar colores de residuos correspondientes

### Iconos Principales del Sistema

#### Navegación
- **Home:** Casa outline
- **Dashboard:** Panel/Layout outline
- **Mapa:** Marcador de ubicación
- **Perfil:** Usuario circle
- **Configuración:** Engranaje

#### Acciones
- **Clasificar:** Cámara / Escáner
- **Guardar:** Diskette / Check
- **Eliminar:** Papelera
- **Editar:** Lápiz
- **Buscar:** Lupa
- **Filtrar:** Embudo
- **Cerrar:** X

#### Categorías de Residuos
- **Plástico:** Botella reciclable
- **Cartón:** Caja
- **Vidrio:** Botella de vidrio
- **Orgánico:** Hoja / Manzana

#### Tecnología
- **IA:** Cerebro / Chip
- **Facial:** Rostro escaneado
- **Ubicación:** Pin de mapa
- **Estadísticas:** Gráfica de barras

### Estilo Visual de Iconos

1. **Consistencia:** Todos los iconos deben tener el mismo stroke width dentro de un tamaño
2. **Simplicidad:** Formas geométricas limpias, evitar detalles excesivos
3. **Alineación:** Centrados en su contenedor, con padding uniforme
4. **Estado hover:** Transición suave a color primario + escala 1.1x

```css
.icon {
  width: 24px;
  height: 24px;
  stroke-width: 2px;
  transition: all 0.2s ease;
}

.icon:hover {
  color: #10b981;
  transform: scale(1.1);
}
```

### Estilo Fotográfico

#### Características
- **Luminosidad:** Imágenes brillantes y claras
- **Saturación:** Colores vibrantes pero naturales
- **Composición:** Limpia, con espacio negativo
- **Tema:** Personas interactuando con tecnología eco-amigable

#### Temas Preferidos
- Personas reciclando con tecnología
- Ciudades verdes e inteligentes
- Naturaleza urbana
- Dispositivos tecnológicos en contextos ecológicos
- Comunidades colaborando

### Tratamiento de Imágenes

#### Overlays
Aplicar overlay gradiente para mejorar legibilidad de texto:
```css
background: linear-gradient(180deg, 
  rgba(0, 0, 0, 0.3) 0%, 
  rgba(0, 0, 0, 0.6) 100%
);
```

#### Filtros
- **Brillo:** +5 a +10%
- **Contraste:** +5 a +15%
- **Saturación:** -5 a +10%
- **Tinte verde sutil:** Opcional en hero images

### Ilustraciones

#### Estilo
- **Línea:** Outline con esquinas redondeadas
- **Color:** Paleta de marca (verdes y azules)
- **Complejidad:** Media, balance entre detalle y simplicidad
- **Contexto:** Tecnología, naturaleza, reciclaje

#### Uso
- Secciones vacías (empty states)
- Onboarding
- Errores 404
- Tutoriales
- Banners informativos

### Imágenes de Productos/Residuos

- Fondo blanco limpio
- Iluminación uniforme
- Alta resolución (mínimo 1200px ancho)
- Formato PNG con transparencia cuando sea posible

## Animaciones y Transiciones

### Principios de Animación

1. **Propósito:** Cada animación debe tener una función (feedback, orientación, deleite)
2. **Rapidez:** Las animaciones deben ser rápidas (150-300ms)
3. **Sutileza:** Preferir animaciones sutiles sobre dramáticas
4. **Consistencia:** Usar las mismas duraciones y easing curves en toda la app

### Duraciones Estándar

| Tipo | Duración | Uso |
|------|----------|-----|
| Instant | 100ms | Cambios de estado hover |
| Quick | 150ms | Botones, checkboxes |
| Base | 200ms | Transiciones estándar |
| Moderate | 300ms | Modales, dropdowns |
| Slow | 500ms | Animaciones complejas |

### Easing Functions

```css
:root {
  --ease-in: cubic-bezier(0.4, 0, 1, 1);
  --ease-out: cubic-bezier(0, 0, 0.2, 1);
  --ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
}
```

## Tono de Comunicación

### Voz de Marca

**EcoVision** comunica con una voz que es:

- *
*Optimista:** Creemos en un futuro sostenible
- **Accesible:** Lenguaje claro, sin tecnicismos innecesarios
- **Empática:** Entendemos que el cambio ambiental es un proceso
- **Innovadora:** Celebramos la tecnología al servicio del planeta
- **Educativa:** Compartimos conocimiento sin ser condescendientes

### Tono por Contexto

#### Landing Page / Marketing
- **Tono:** Inspirador, motivador, optimista
- **Estilo:** Frases cortas, llamados a la acción claros
- **Ejemplo:** "Revoluciona tu manera de reciclar. Inteligencia artificial que cuida el planeta."

#### Dashboard / App
- **Tono:** Profesional, directo, eficiente
- **Estilo:** Instrucciones claras, feedback inmediato
- **Ejemplo:** "Clasificación completada. Has reciclado 15 items esta semana."

#### Errores y Validaciones
- **Tono:** Empático, constructivo, orientado a soluciones
- **Estilo:** Explicar el problema y ofrecer solución
- **Ejemplo:** "No pudimos procesar tu imagen. Asegúrate de que haya buena iluminación e intenta nuevamente."

#### Onboarding / Tutoriales
- **Tono:** Amigable, guía, motivador
- **Estilo:** Pasos claros, celebrar progreso
- **Ejemplo:** "¡Excelente! Ya configuraste tu perfil. Ahora probemos el clasificador de residuos."

### Mensajes de Éxito

✅ **Hacer:**
- "¡Genial! Tu residuo fue clasificado correctamente."
- "Guardado exitosamente. Contribuiste a un planeta más limpio."
- "Perfecto. Tu perfil ha sido actualizado."

❌ **Evitar:**
- "La operación fue exitosa." (demasiado técnico)
- "ÉXITO!!!" (demasiado exagerado)
- "Todo bien" (vago, poco informativo)

### Mensajes de Error

✅ **Hacer:**
- "No pudimos conectar con el servidor. Verifica tu conexión e intenta nuevamente."
- "El formato de email no es válido. Ejemplo: usuario@dominio.com"
- "La contraseña debe tener al menos 8 caracteres."

❌ **Evitar:**
- "Error 500: Internal Server Error" (técnico, no ayuda al usuario)
- "Algo salió mal" (vago, genera frustración)
- "Tu culpa que no funciona" (acusatorio)

### Llamados a la Acción (CTAs)

**Formato:** Verbo + Beneficio

✅ **Efectivos:**
- "Comenzar a Reciclar"
- "Clasificar mi Residuo"
- "Ver mi Impacto Ambiental"
- "Encontrar Puntos Cercanos"
- "Crear mi Cuenta"

❌ **Débiles:**
- "Haz clic aquí"
- "Continuar"
- "Enviar"
- "OK"

### Copywriting Guidelines

1. **Claridad sobre creatividad:** La función primero
2. **Activo sobre pasivo:** "Clasifica tus residuos" vs "Los residuos son clasificados"
3. **Específico sobre genérico:** "Ahorra 2kg de CO₂" vs "Ayuda al planeta"
4. **Beneficio sobre característica:** "Encuentra puntos de reciclaje en 5 segundos" vs "Sistema de búsqueda rápido"
5. **Humano sobre corporativo:** "Te ayudamos a" vs "La plataforma facilita"

### Lenguaje Inclusivo

- Usar "persona usuaria" o "usuario/a" cuando sea apropiado
- Evitar genéricos masculinos cuando hay alternativas neutras
- Preferir términos como "comunidad", "personas", "equipo"
- Representar diversidad en imágenes y ejemplos

### Términos Clave de la Marca

**Usar:**
- Clasificación inteligente
- Reconocimiento facial
- Punto de reciclaje
- Impacto ambiental
- Ciudad inteligente

**Evitar:**
- Basura (usar "residuo")
- Botar (usar "reciclar" o "desechar correctamente")
- App (preferir "plataforma")

## Elementos Prohibidos

### ❌ No Hacer

#### Logotipo
1. ⛔ Rotar o inclinar el logotipo
2. ⛔ Cambiar colores no autorizados (rosado, morado, naranja fuerte)
3. ⛔ Aplicar efectos 3D, sombras drop shadow, o bevel
4. ⛔ Deformar o estirar el logotipo
5. ⛔ Usar sobre fondos que comprometan legibilidad
6. ⛔ Añadir elementos decorativos alrededor
7. ⛔ Rediseñar o "mejorar" el logotipo

#### Tipografía
1. ⛔ Usar más de 3 fuentes diferentes en una página
2. ⛔ Outline o stroke en texto
3. ⛔ All caps en párrafos largos
4. ⛔ Texto justificado
5. ⛔ Tracking excesivo (> 0.1em)
6. ⛔ Leading menor a 1.2x el tamaño de fuente

#### Colores
1. ⛔ Inventar nuevos colores fuera de la paleta
2. ⛔ Usar degradados rainbow o multicolor
3. ⛔ Combinaciones de bajo contraste para texto
4. ⛔ Colores neón o fosforescentes

#### Composición
1. ⛔ Centrar todo el contenido
2. ⛔ No respetar los márgenes y padding
3. ⛔ Sobresaturar con demasiados elementos
4. ⛔ Usar Comic Sans o Papyrus (nunca)
5. ⛔ Clipart genérico o imágenes de baja calidad
6. ⛔ Más de 2 CTAs en una misma sección

#### Animaciones
1. ⛔ Animaciones que duran más de 1 segundo
2. ⛔ Movimientos bruscos o jarring
3. ⛔ Autoplay de videos con sonido
4. ⛔ Efectos de parallax excesivos que marean
5. ⛔ Carruseles automáticos (ux antipattern)
